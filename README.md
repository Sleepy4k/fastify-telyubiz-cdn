# Fastify Telyubiz CDN

A Content Delivery Network (CDN) service built with Fastify and TypeScript.

## Description

This project provides a high-performance CDN solution using Fastify framework, designed to serve static assets efficiently.

## Features

- Fast static file serving
- Built with TypeScript for type safety
- Powered by Fastify for high performance
- Scalable architecture

## Installation

```bash
bun install
```

## Usage

```bash
# Development
bun dev

# Build
bun run build

# Production
bun start
```

## Configuration

Configure your CDN settings in the configuration file.

## Tech Stack

- **Framework**: Fastify
- **Language**: TypeScript
- **Runtime**: Node.js
